# COP3530_Project_3_Group_5

HOW TO RUN:

Download Visual Studio Community 2022 (Link here: https://visualstudio.microsoft.com)
Install anything that says C++ in Title, should be ~16.6 Gb
Download GitBash (Link here: https://git-scm.com/downloads)
Clone repo, open CMakeLists.txt, and wait for it to run.
Set the startup button to bin\main.exe

If you are updating to a new pull: Delete the out file folder and rebuild the project by ctrl+S on CmakeLists.txt
